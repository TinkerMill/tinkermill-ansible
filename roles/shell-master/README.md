shell-master
=========

A role to configure a system so it acts as a great shell system.  For example, running scripts, interacting with git & ansible, etc.

Requirements
------------

None

Role Variables
--------------

None

Dependencies
------------

None

Example Playbook
----------------

    - hosts: servers
      roles:
         - { role: shell-master }
